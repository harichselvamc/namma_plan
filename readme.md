# Namma Plan

Namma Plan is a simple open-source Android app built with **Kotlin** and **Jetpack Compose**.

It helps families plan:

* Festivals
* Pilgrimages
* Trips
* Savings goals
* Important events

## Features

* Event countdown
* Savings tracker
* Checklists
* Offline reminders
* Tamil and English
* Backup and restore
* Works without an account
* No developer-owned server

The app works offline and stores data on the user’s device.

Users can choose General, or custom event templates.

Full features, versions, and development plans are available in the GitHub Wiki.
